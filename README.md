# BrokerReport

Анализ брокерских отчётов (HTML) с веб-интерфейсом и SQLite.

## Быстрый старт

```powershell
# Установка зависимостей
pip install flask beautifulsoup4 lxml

# Запуск
python run.py
```

Откройте http://127.0.0.1:5000

## Как это работает

1. HTML-отчёты брокера кладутся в папку `reports/`
2. При запуске скрипт автоматически импортирует все `.html`/`.HTML` файлы в SQLite
3. Фоновый дозор раз в минуту проверяет новые файлы
4. На дашборде — прибыль/убыток по трейд-сделкам (LIFO), открытые позиции, расходы на РЕПО
5. Фильтр по датам позволяет выбирать период для анализа

## Структура

```
├── copilot/instructions.md   # инструкции для Copilot-агента
├── app/                      # исходный код (Flask, парсер, БД)
├── reports/                  # HTML-отчёты брокера
├── run.py                    # точка входа
└── broker.db                 # SQLite (создаётся автоматически)
```
